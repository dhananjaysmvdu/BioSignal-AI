- Implemented `scripts/ethics_engine/run_ethics_compliance_engine.py`:
  - Loads fairness metrics from `observatory/fairness_metrics.csv`
  - Computes **Bias Score (BS)** = |max(group_mean) - min(group_mean)| / mean(global)
